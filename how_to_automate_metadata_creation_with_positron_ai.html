How to Automate Metadata Creation with Positron AI

This task guides users through the process of generating metadata for a DITA topic using Positron AI.

  1. Open a DITA topic in Oxygen XML Editor
    Ensure the topic you want to update with metadata is open in the editor.
  2. Access the Positron AI Assistant
    Click on the Positron AI Assistant icon in the toolbar to open the assistant panel.
  3. Select "Generate Metadata"
    From the assistant options, choose "Generate Metadata." The AI will provide suggestions based on the topic content.
  4. Review and Apply Metadata Suggestions
    Review the generated metadata, such as keywords and topic categories. Apply the suggestions or modify them as needed.
