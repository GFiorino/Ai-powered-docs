How to Use Positron AI for Documentation.

This guide explains how to use Positron AI in Oxygen XML Editor to streamline technical writing tasks.

  1. Open Oxygen XML EditorEnsure Positron AI is enabled in your workspace.
  2. Start a New DITA TopicGo to File → New → DITA → Topic and select the appropriate template (Concept, Task, etc.).
  3. Use Positron AI for Metadata SuggestionsClick on the Positron AI Assistant icon and select "Generate Metadata." Review the suggestions and apply them as needed.
  4. Leverage Content SuggestionsHighlight a section of your topic and ask Positron AI to suggest content or refine existing text.
