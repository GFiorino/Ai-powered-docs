Comparison of AI Tools for Documentation

This task explains how to use Positron AI to validate a DITA topic for structural, grammatical, and formatting errors.

  1. Open the DITA topic in Oxygen XML Editor
    Ensure that the topic you want to validate is open in the editor.
  2. Access the Positron AI Validation Tool
    Click on the Positron AI Assistant icon and select "Validate Topic."
  3. Review the Validation Report
    Analyze the errors and warnings provided by Positron AI. Common issues include structural errors, grammatical mistakes, and metadata inconsistencies.
  4. Correct the Issues
    Use the suggestions in the validation report to fix the identified issues directly in the editor.
